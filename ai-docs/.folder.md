# /ai-docs

这里承接 AI 生成的附属文档，不承接代码实现本身。
根目录之外的新增 Markdown 说明、计划、总结都应落在这里。
如果这里开始分子类目录，必须继续为子目录创建 `.folder.md`。

- `.folder.md`：本目录的用途说明与输出边界锚点。
