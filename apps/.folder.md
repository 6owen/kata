# /apps

这里放仓库内的示例应用或宿主应用占位。
当前包含浏览器扩展与 Web 应用两个入口。
后续新增 app 类型时，必须在这里先登记职责边界。

- `.folder.md`：本目录的局部索引与同步锚点。
- `browser-extension/`：浏览器扩展应用目录，细节由子目录 `.folder.md` 继续描述。
- `web/`：Web 应用目录，细节由子目录 `.folder.md` 继续描述。
