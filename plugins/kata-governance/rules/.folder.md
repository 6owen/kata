# /plugins/kata-governance/rules

这里存放 `kata-governance` 的规则源码，而不是目标仓库里的运行结果。
bootstrap skill 会按固定顺序读取这里的规则文件，并生成目标项目的 `AGENTS.md`。
新增、拆分或合并规则时，必须保持这里与安装脚本一致。

- `.folder.md`：本目录的局部索引与同步锚点。
- `general.md`：跨项目通用的 AI 工作规则，含范式复用清单与品味自检。
- `frontend.md`：前端样式与设计 token 复用相关规则。
- `repo-docs.md`：三层文档结构、同步协议、回环工作流、违规分级、Header 协议与输出路径规则。
