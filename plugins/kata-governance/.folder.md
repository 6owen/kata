这里承载仓库治理规则与安装型 skill 的 plugin 根目录。
README 负责解释治理定位，`plugin.json` 负责发布元数据，`skills/` 负责具体安装能力。
治理规则如果变化，必须先在这里收口。

- `.folder.md`：本目录的局部索引与同步锚点。
- `README.md`：面向人类的 plugin 定位说明。
- `plugin.json`：JSON 例外文件；提供 `kata-governance` 的发布元数据。
- `rules/`：通用 AI 工作规则源码目录，供 bootstrap 生成目标仓库 `AGENTS.md`。
