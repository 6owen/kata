# starter-react

Use `6owen/starter-react` as the baseline for new TanStack Start + React single-app projects, or as the target structure when normalizing an existing single-package frontend, including `src/pages` route groups, `route.tsx` and `index.tsx` file routing, page-private `-components`, `services` / `stores` / `setups` ownership, token-first Tailwind styling, and a vendored `react-best-practices` companion reference.

Quick init entry: `scripts/apply-starter-react.mjs`

Main entry: `SKILL.md`
