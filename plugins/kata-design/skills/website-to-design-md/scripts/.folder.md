# /plugins/kata-design/skills/website-to-design-md/scripts

这里放置 `website-to-design-md` 的可执行脚本。
当前覆盖浏览器工具检测、网站证据抽取与设计预览渲染三类任务。
脚本参数、默认文件名或解析规则变化时必须同步回写文档。

- `.folder.md`：本目录的局部索引与同步锚点。
- `check-browser-tooling.mjs`：检测 `agent-browser`、Node 与 npm 可用性。
- `extract-browser-evidence.mjs`：从实时网站抽取 DOM、样式与文本证据。
- `render-design-preview.mjs`：把 `DESIGN.md` 渲染成配套 HTML 预览。
