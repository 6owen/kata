# /plugins

这里承载仓库的一方 plugin 实现与骨架。
每个 plugin 都必须能独立暴露自己的 README、`.folder.md`、`plugin.json` 与 skills 结构。
后续新增 plugin 时，默认继承根目录 AI 约束协议。

- `.folder.md`：本目录的局部索引与同步锚点。
- `kata-code/`：代码工程与工具链相关 plugin。
- `kata-design/`：设计流程编排相关 plugin。
- `kata-governance/`：仓库治理规则与安装型 bootstrap skill plugin。
- `kata-test/`：测试相关 plugin 骨架。
